# Binary Search Game


Простая реализация игры «Угадай число» с использованием алгоритма бинарного поиска.


## Описание


Программа моделирует процесс угадывания числа от 1 до 100 методом бинарного поиска. Функция `game_core_v3` принимает загаданное число и возвращает количество попыток, потребовавшихся для его нахождения.


## Особенности


- Алгоритм бинарного поиска: O(log n).
- Диапазон чисел: 1–100.
- Подсчёт количества итераций.
- Соответствие PEP 8.
- Зависимости в `requirements.txt`.
